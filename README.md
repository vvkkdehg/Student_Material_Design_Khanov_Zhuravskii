## Создание приложения «Список студентов» с использованием Material Design

### Описание проекта
Приложение демонстрирует возможности **Material Design 3** с использованием **Jetpack Compose**. В нём реализован список студентов с возможностью раскрытия подробной информации об каждом, с поддержкой светлой и тёмной темы, кастомной типографикой, формами и плавными анимациями.

## Цель работы

Изучить и применить на практике принципы **Material Theming** в Android-приложениях с помощью **Jetpack Compose**, включая:
- настройку цветовой схемы;
- поддержку светлой и тёмной темы;
- создание кастомной типографики;
- использование различных форм;
- добавление анимаций для улучшения пользовательского опыта.

## Функциональность

Приложение предоставляет следующий набор возможностей:

- Отображение списка студентов.
- Раскрывающееся описание каждого студента с юмористическим контентом.
- Анимированное раскрытие и сворачивание карточки.
- Кастомная цветовая схема для светлой и тёмной темы.
- Поддержка **Material Design 3** компонентов.
- Круглые аватары студентов.
- Скруглённые углы карточек с диагональным эффектом.
- Использование пользовательских шрифтов **Montserrat** и **Abril Fatface**.
- Полноценная адаптация под тёмную тему.

## Технологии и библиотеки

- Kotlin
- Jetpack Compose
- Material Design 3
- Google Fonts
- Material Icons Extended
- Compose Animation API
- Gradle
